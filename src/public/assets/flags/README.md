# World Cup sweepstake flag assets

Tiny SVG flag assets for the 48-team sweepstake app.

Folders:
- `24x18/` for group cards and tables
- `20x15/` for dense rows and fixtures

Use `flagRegistry.json` to map country names to files.

Note: complex coats of arms, text and crests are simplified because these assets are intended for very small UI sizes. This avoids unreliable emoji rendering across operating systems.
